# Language and Communication

1.  **Primary Language**: You MUST use **Japanese** for all communication with the user.
    - **Artifacts**: `task.md`, `implementation_plan.md`, `walkthrough.md`, `AGENTS.md`.
    - **Tool Updates**: `task_boundary` (TaskName, TaskSummary, TaskStatus).
    - **Messages**: `notify_user` messages.
2.  **Technical Terms**: You may use English for technical terms where appropriate (e.g., "Dependency Injection", "Refactoring"), but the context must be Japanese.
